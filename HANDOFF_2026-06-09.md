## 1. プロダクト概要

**ソコナラ** = 地域キャリア共創プラットフォームMVP。
地域企業の経営層の記事を読む → 価値観アンケート(4問) → AI診断「はたらくヒント」 → 経営層との接点（WEBCASフォーム）。

### ファイル
- `index.html` … ユーザー画面SPA（3300行超）
- `admin.html` … 管理画面
- `api/feedback.js` … AI診断（OpenAI gpt-4o for values / gpt-4o-mini for others）
- `api/line-token.js` … LINE OAuth
- `api/interest-forms.js` … 興味フォーム管理（service_role）
- `api/report.js` … 共感/診断ログの集計・削除（service_role、table指定対応）
- `api/line-push.js` … LINE配信（管理パスワード必須）
- `api/check-friend.js` … 未使用（dead）
- `api/admin-auth.js` … 管理ログイン
- `legal/terms.pdf` `legal/privacy.pdf` … 公式法務PDF（直接配信）
- `manifest.json` … PWA（id/scope/maskable対応済）
- `images/` … logo-02.png, icon-192/512/1024.png

## 2. ユーザー画面の流れ（index.html）

```
p-list（経営層一覧・TOP）
  ↓ 経営層カード/顔写真クリック → openCompanyArticle()
p1（記事）
  ↓ 「はたらくヒントを受け取る（無料）」
p2（アンケート Q1〜Q4）
  ↓ submitEmpathy
p3（AI診断「はたらくヒント」）
  ↓ LINE未登録 → 途中で「…」カット＋LINE登録CTA
  ↓ LINE登録済 → フル表示 ＋「接点を作る」
p4（おすすめ経営層）／ p5（接点：WEBCASフォームへ）
```

### 状態 S
`const S = { q1, q2(配列), about(配列), q3, free, fb, screen, mode, adminTab }`
- Q1=刺さった言葉(記事から1つ) / Q2=求めるもの(複数) / Q3(about)=今の状況(複数) / Q4=自由記述
- **Q3の旧「関わり方」質問は削除済み**（S.q3は常に''、WEBCASで聞く）

## 3. AI診断（api/feedback.js の values）— 調整履歴

- **モデル**：values は `gpt-4o`、その他は `gpt-4o-mini`。max_tokens: values=1800。
- **占い/胡散臭さ対策**：ユーザーが選んだ回答を根拠として明示してから解釈。「言い当ててやろう」調禁止。
- **「かもしれません」は最大2回まで**。
- **主語の取り違え厳禁**（経営層の記事/経歴をユーザーの経験として書かない）。
- **平易・具体的**（抽象的美文禁止）。**長さ可変**（自由記述が濃いほど長く最大30行・水増し禁止）。
- **wantedProfile** をマッチング材料に（求人票化禁止・人柄相性として1文）。
- **summary は5〜6文**（未登録プレビューで途中カットするため）。

### 表示（showFeedback）
- `isLineRegistered()`（snLineId有無）で分岐。未登録：summaryを165文字で文の途中カット＋「…」＋LINE登録CTA。登録済：フル詳細＋接点CTA。
- values（価値観キーワード）表示は**廃止**（values:[]）。
- 結果は `sn_fb_cache`（localStorage 24h）。記事再訪で「前回の診断結果を見る」ボタン。

## 7. 管理画面（admin.html）

- セッション：`sessionStorage.snAdminPw`＋`localStorage.snAdmin`(24h)。`_adminPw`でAPI認証。
- **記事管理（art-*フォーム）**：Notion貼付→解析→セクション編集。メタに「会社一言紹介」「記事冒頭コメント(summaryComment)」「こんな人と出会いたい(wantedProfile)」「タグ」等。保存=saveArticle()。
  - ⚠️ **旧「企業登録(addco/co-*)」フォームも併存**（業種・規模・手数料・会社名編集はこちらだけ）。両方残す方針(案A)。
- **興味通知フォーム**：一覧＋CSV＋個別削除＋**選択して一括削除**。
- **AIレポート生成タブ**：①全経営層サマリー(❤共感数・📋診断数の表) ②❤共感ログ(いつ/どの記事) ③📋診断ログ(いつ/どの記事/刺さった言葉/コメント) ④経営層別詳細(分布バー＋匿名コメント[時刻付]＋CSV＋テストデータ消去)。各ログに個別削除＋全消去。
- **LINE配信**：broadcast/push（管理パスワード必須）。
